#Tarea 2 - Ejercicios con setInterval

Alumno: Kilian Rodriguez Duque

Descripción

Este repositorio contiene 5 ejercicios prácticos de JavaScript centrados en el uso de funciones temporizadas como setInterval y clearInterval.

Cada ejercicio está implementado en un archivo HTML independiente.

Ejercicios
Ejercicio 1: Reloj en tiempo real

Archivo: ejercicio1.html
Muestra la hora actual en tiempo real, actualizándose automáticamente cada segundo al cargar la página.

Ejercicio 2: Cuenta atrás con parada

Archivo: ejercicio2.html
Realiza una cuenta atrás desde 10 hasta 0. Incluye botones para iniciar y detener el conteo. Al llegar a 0 muestra "¡Tiempo!".

Ejercicio 3: Texto parpadeante

Archivo: ejercicio3.html
Un texto alterna entre visible e invisible cada 500 ms. Incluye botones para iniciar y detener el efecto.

Ejercicio 4: Barra de progreso animada

Archivo: ejercicio4.html
Una barra de progreso se llena del 0% al 100% en 10 segundos. Al completarse muestra "¡Completado!".

Ejercicio 5: Parar el número

Archivo: ejercicio5.html
Muestra un número aleatorio entre 1 y 10 que cambia rápidamente. El usuario debe detenerlo en el número 7 para ganar.
